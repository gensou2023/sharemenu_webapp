## 第1部: プロダクトビジョン（PdO）

### 1.1 ミッション

> **「すべての飲食店オーナーに、プロ品質のメニューデザインを、チャットするだけで届ける。」**

### 1.2 ビジョン

> **「飲食店のビジュアル格差をなくし、すべてのお店が"映える"メニューで集客できる未来を創る。」**

### 1.3 提供する価値（Value Proposition）

| 価値の柱 | 詳細 | 従来との比較 |
|----------|------|-------------|
| **コスト削減** | メニューデザイン外注費（1件3-10万円）をほぼゼロに | 外注: 数万円/件 → 月額700円で無制限 |
| **時間短縮** | 外注で数日かかっていた制作を5-10分で完了 | 外注: 3-7日 → 5-10分 |
| **専門知識不要** | チャットで日本語を打つだけ | デザインツール不要 |
| **SNSマルチ対応** | Instagram(1:1, 9:16)、X(16:9)の3サイズを同時生成 | 手動リサイズ不要 |
| **即時更新** | 季節メニュー・日替わりの変更にリアルタイム対応 | 再外注不要 |
| **プロ品質提案** | AIがキャッチコピー・ハッシュタグまで提案 | マーケティング知識不要 |

## 第10部: リスクと対策（統合）

### 10.1 ビジネスリスク

| リスク | 影響度 | 対策 |
|--------|--------|------|
| **Free→Pro転換率が想定以下** | 高 | Freeの制限を適切に設計。Proトライアル7日間の導入検討。3サイズプレビュー（透かし入り）で価値を実感させる |
| **ターゲットへのリーチ困難** | 高 | 飲食業界メディア連携（食べログ等）。展示会出展。導入事例の口コミプログラム。LINE公式アカウント導線 |
| **競合の参入・機能追加** | 中 | 飲食店×日本語×チャットUXのニッチ特化。蓄積データによる独自プロンプト最適化 |
| **解約率（Churn）が高い** | 中 | SNS投稿カレンダー機能で継続利用促進。年間プラン割引で長期契約誘引 |

### 10.2 技術リスク

| リスク | 影響度 | 対策 |
|--------|--------|------|
| **日本語テキストが崩れる** | 高 | 「コンセプトデザイン」と位置づけ、テキスト情報は別途提供。ポストプロセスでのオーバーレイ合成を検討 |
| **画像生成APIのレート制限・障害** | 高 | リクエストキュー（非同期処理）。プログレスバー+推定待ち時間表示。フォールバックプロバイダー検討 |
| **Gemini APIの応答品質ブレ** | 中 | Few-shot例充実。出力フォーマットのバリデーション。継続的プロンプト改善 |
| **Streamlitのスケーラビリティ限界** | 中 | 100ユーザー超過時にNext.js移行開始。セッション管理のステートレス化を早期設計 |
| **著作権・不適切コンテンツ** | 中 | 利用規約に明記。コンテンツモデレーション導入検討。商用利用ライセンス条項の明確化 |
